Site Crawler

This program is being written for CPS 470 Computer Networks. At this time the program only uses 100 threads to "crawl" 100 urls from a text file. Crawling in this case consists of connecting the the hosts, sending a HEAD request, and checking the response for presence of a robots.txt file. If one is not present the program will move forward with sending a GET request. Responses are printed to the console from all requests sent. By the completion of the project, the program will "crawl" one millions URLs, with an increased thread count to greatly speed up the process.

Language: Python
